# SiteGuard365 Dashboard Suite

A Stripe-style client portal and staff dashboard for SiteGuard365, powered by REST APIs and custom database tables.

## Setup

1. Upload the `sg365-dashboard-suite` folder to `/wp-content/plugins/`.
2. Activate the plugin in WordPress.
3. Create the demo pages:
   - **Client Portal** page with `[sg365_client_app]`
   - **Staff Dashboard** page with `[sg365_staff_app]`
4. WooCommerce users can access `/my-account/sg365-portal` when WooCommerce is active.
