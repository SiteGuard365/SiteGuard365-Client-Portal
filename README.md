# SiteGuard365 Dashboard Suite
Stripe-style client portal and staff dashboard with REST-first data storage, custom tables, and WooCommerce My Account integration.

## Setup
1. Upload the plugin folder to `/wp-content/plugins/` and activate.
2. Ensure WooCommerce is active to enable the My Account endpoint.
3. Visit **SG365 Dashboard** in the admin menu to configure branding and menus.

## Demo Pages
* Create a page named **Client Portal** with shortcode `[sg365_client_app]`.
* Create a page named **Staff Dashboard** with shortcode `[sg365_staff_app]`.
* My Account endpoint: `/my-account/sg365-portal`.
